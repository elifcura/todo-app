 # React Todo App

Bu proje React ve Vite kullanılarak geliştirilmiş basit bir Todo (Yapılacaklar) uygulamasıdır.

## 🚀 Özellikler

- Todo ekleme
- Todo silme
- Responsive tasarım
- Netlify ile canlı yayın

## 🛠 Kullanılan Teknolojiler

- React
- Vite
- JavaScript
- Netlify
